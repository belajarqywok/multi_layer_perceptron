✈ Plane Shooter

Enemy AI powered | C++ (CMake) - Multi Layer Perceptron (MLP) - ONNX - Javascript inference

🧠 How the AI Works

Each enemy feeds 5 game-state features into the trained MLP every 20 frames. The output is an aggression score [0–1] that controls shooting frequency, bullet dodging, and player tracking.

⚙️ ML Pipeline

  1. Generate dummy data (Python script)
  2. Train MLP engine (C++17, from scratch)
  3. Export weights to ONNX via Protobuf
  4. Extract weights → model_weights.js
  5. Pure JS forward pass in model.js
